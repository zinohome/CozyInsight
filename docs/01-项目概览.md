# DataEase 项目概览

## 项目简介

**DataEase** 是一个开源的商业智能(BI)工具,致力于帮助用户快速分析数据并洞察业务趋势,从而实现业务的改进与优化。

### 核心定位

- **产品类型**: 开源 BI 数据可视化平台
- **目标用户**: 企业数据分析师、业务人员、IT 管理员
- **核心价值**: 零门槛、简单易用、全场景支持、安全分享、AI 增强

### 主要特性

1. **开源开放**: 
   - GPL-3.0 许可证
   - 按月迭代更新
   - 社区活跃(GitHub 高星项目)

2. **简单易用**:
   - 极易上手
   - 通过拖拉拽方式创建图表
   - 所见即所得的可视化编辑器

3. **数据源支持广泛**:
   - OLTP 数据库: MySQL、Oracle、SQL Server、PostgreSQL、MariaDB、Db2、TiDB、MongoDB-BI等
   - OLAP 数据库: ClickHouse、Apache Doris、Apache Impala、StarRocks等
   - 数据仓库/数据湖: Amazon RedShift等
   - 数据文件: Excel、CSV等
   - API 数据源

4. **全场景支持**:
   - 桌面版 (PC 应用)
   - 服务器版 (Web 应用)
   - 移动端适配
   - 多种嵌入方式

5. **AI 加持**:
   - 集成 SQLBot 实现智能问数
   - AI 驱动的数据分析

### 版本信息

- **当前版本**: v2.10.17
- **Java 版本**: Java 21
- **Spring Boot 版本**: 3.3.0
- **Vue 版本**: Vue 3.3.4

### 部署模式

1. **桌面版 (Desktop)**:
   - 单机应用
   - 内置 H2 数据库
   - 适合个人使用

2. **单机版 (Standalone)**:
   - 默认部署模式
   - 集成所有功能
   - 适合中小企业

3. **分布式版 (Distributed)**:
   - 企业版
   - 支持横向扩展
   - 适合大型企业

### 项目规模

#### 代码规模
- **后端 Java 文件**: ~496 个
- **前端 Vue 组件**: 大量 Vue 组件(估计数百个)
- **数据库表**: 约 50+ 核心表
- **数据库 DDL**: ~4250 行

#### 模块划分
- **后端模块**: 27 个主要业务模块
- **前端视图**: 24 个主要页面模块

### 技术架构

采用前后端分离的现代化架构:

**后端**:
- Spring Boot 3.3.0 (Java 21)
- MyBatis Plus (ORM)
- Apache Calcite (SQL 引擎)
- Apache SeaTunnel (数据集成)
- H2/MySQL (数据存储)

**前端**:
- Vue 3.3.4 (渐进式框架)
- Element Plus (UI 组件库)
- AntV (数据可视化图表库)
- Vite (构建工具)

**基础设施**:
- Docker (容器化部署)
- Redis (缓存)
- WebSocket (实时通信)

### 核心功能领域

1. **数据连接与管理**
   - 数据源连接
   - 数据集管理
   - 数据同步与调度

2. **数据可视化**
   - 图表编辑器
   - 仪表板设计
   - 可视化大屏

3. **数据分析**
   - SQL 查询引擎
   - 计算字段
   - 数据过滤与钻取

4. **权限与安全**
   - 用户权限管理
   - 数据权限控制
   - 水印与安全分享

5. **系统管理**
   - 系统配置
   - 任务调度
   - 导出中心
   - 消息通知

## 总结

DataEase 是一个成熟的企业级 BI 平台,具有以下特点:

✅ **技术栈现代化**: 采用 Spring Boot 3.x + Vue 3 架构
✅ **功能完整**: 涵盖数据连接、可视化、分析、权限等全流程
✅ **架构清晰**: 模块化设计,职责分明
✅ **可扩展性强**: 支持多种部署模式和数据源
✅ **社区活跃**: 持续迭代更新

该项目适合作为企业级 BI 解决方案,也是学习 BI 系统架构的优秀案例。
